## 安装文件

### 下载

推荐在国内源下载。[传送门](https://mirrors.tuna.tsinghua.edu.cn/apache/tomcat/)

### 启动

- 启动

```sh
cd /your_path/tomcat/apache-tomcat-8.5.28/bin
./catalina.sh start
```

- 停止

```sh
cd /your_path/tomcat/apache-tomcat-8.5.28/bin
./catalina.sh stop
```
