# 🍔 准备安装介质

警告

数据无价，注意备份重要数据。

准备一个可以格式化的 U 盘。可以在任何系统中制作安装介质。

# windows

在 window 下，可以选择 rufusopen in new window 制作镜像。推荐下载便携版，开箱即用。

  • 设备：需要刻录的 U 盘

  • 引导类型选择：下载的 ISO 镜像

也可以同样安装 Etcheropen in new window，具体使用参靠下面说明。

# macOS

在 mac 下，推荐使用 Etcheropen in new window，可以在官网下载，如果使用 brew 管理软件，可以通过如下命令：

brew install balenaetcher

# Ubuntu

在 Ubuntu 中套娃制作启动盘，可以直接使用默认安装的程序 制作启动盘 刻录安装镜像。

当然也可以同样安装 Etcheropen in new window，这是一个跨平台程序。

curl -1sLf 'https://dl.cloudsmith.io/public/balena/etcher/setup.deb.sh' | sudo -E bash

sudo apt update
sudo apt install balena-etcher-electron

# 🍕 电脑 BIOS 设置

  • 关闭 Security Boot

  • 打开虚拟化（推荐）

    • Intel Virtual Technology（Intel）
