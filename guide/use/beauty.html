📝 写在最前

每个人使用 Ubuntu 的目的不同。一些人热衷于折腾桌面，以获取使用上的体验；有些人只是作为服务器使用，不对界面有要求。怎么折腾，都是每个人自己的选择。

我不太在意使用的桌面，这里只是简单美化一下。

如果你也一样不太在意这个，可以不在美化上花费过多的时间。少折腾系统配置，多折腾效率工具，系统只是个工具。

Gnome 桌面有一个专门的主题网站，可以在这里找到非常多优秀的主题：https://www.gnome-look.orgopen in new window

# gnome-tweaks 主题管理工具 
sudo apt install -y gnome-tweaks

下面的内容只是简单介绍一下只美化主题、图标、鼠标指针等。

警告 🚧

如果你是新手，建议尽量不要美化 GRUB，以免引起不必要的麻烦。

# 🌷 Gnome 扩展

安装扩展可以方便我们的使用。你可以在 Gnome extensionsopen in new window 查看到所有的 Gnome 扩展。

  • 安装插件本地连接器open in new window
sudo apt install chrome-gnome-shell

💡 提示

在不久的将来，chrome-gnome-shell 会被更名为 gnome-browser-connector，请参考 wikiopen in new window

  • 安装浏览器扩展插件

在 Chromeopen in new window 中安装。

如果你没有合适的网络在 Chrome 中安装插件，可以在 Firefox 中安装。

在 Firefoxopen in new window 中安装。

  • 根据需求安装插件

在 PPA 中，也内置了一部分常用的插件，你可以通过 sudo apt install gnome-shell-extensions 安装。（注意：通过该方式安装过的插件，无法下面的方式更新）

在 Gnome extensionsopen in new window 可通过网页上的开关按钮，轻松添加插件。插件将会被安装在：~/.local/share/gnome-shell/extensions

这里推荐一些常用的插件，请按需选择：

# 💐 主题

将下载的主题移动到 ${HOME}/.themes 文件夹下，即可通过 gnome-tweaks 选择自定义主题。

# 🌺 图标

将下载的图标主题移动到 ${HOME}/.icons 文件夹下，即可通过 gnome-tweaks 选择自定义图标。

推荐一个常用的图标主题 🦶 papirus-icon-themeopen in new window，可以通过 PPA 进行安装。

sudo add-apt-repository ppa:papirus/papirus
sudo apt-get update
sudo apt-get install papirus-icon-theme

# 🌼 鼠标指针

将下载的鼠标指针主题移动到 ${HOME}/.icons 文件夹下，即可通过 gnome-tweaks 选择自定义鼠标指针。

# 🌼 Shell 主题

Shell 主题是对顶栏和 dock 栏配置主题。

如果出现 ⚠ 无法设置，请按上文安装 Gnome 扩展，并打开 User Themeopen in new window 扩展。
