# 🥛 使用英文目录

如果使用中文安装，用户目录下的目录名称都是中文的，如：桌面，文档，下载 等等，如果有终端操作，切换目录还是挺麻烦的，推荐将其改成英文的。

可以使用如下命令更改：

export LANG=en_US
xdg-user-dirs-gtk-update

更改后，可能会有弹窗询问是否要根据系统语言更改目录，选择保留英文即可。

# 🍷 Gnome 插件

本指南主要是介绍 Gnome 桌面的 Ubuntu 系统。该插件是 Gnome 桌面下的插件。

为有一个更加易用的系统，安装插件可以方便我们的使用。

这里介绍使用使用浏览器安装扩展插件，虽然在 apt 中也能安装，但是只能获取到一部分，且版本较老。

你可以在 Gnome extensionsopen in new window 查看到所有的插件。

  • 安装插件本地连接器
sudo apt install chrome-gnome-shell
  • 安装浏览器扩展插件

Chromeopen in new window

sudo apt install gnome-tweak

插件的安装位置在：~/.local/share/gnome-shell/extensions

# 🍾 终端

配置好一个终端，可已让我们在使用的过程中更加方便。我通常会使用 oh-my-zshopen in new window。

当然有很多选择如：fishshellopen in new window，powerlevel10kopen in new window，可以选择自己喜欢的进行配置。这里主要介绍 oh-my-zsh 的安装。

# 安装 zsh
sudo apt install zsh

# oh-my-zsh
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
# 或
sh -c "$(wget https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh -O -)"

# 更改默认终端为 zsh
sudo chsh -s $(which zsh)

安装完成后，可以通过 ~/.zshrc 配置主题，插件，环境变量等内容。

# oh-my-zsh 插件

oh-my-zsh 也可以安装插件来获取更好的使用体验，插件推荐可参考：传送门open in new window。

安装插件的主要方式是克隆相应的仓库到 oh-my-zsh/custom/plugins 下，并在配置文件中配置。以下为举例：

# 获取插件
git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions

git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting

# 在 .zshrc 中配置
plugins=(
     git
     extract
     zsh-autosuggestions
     zsh-syntax-highlighting
)
